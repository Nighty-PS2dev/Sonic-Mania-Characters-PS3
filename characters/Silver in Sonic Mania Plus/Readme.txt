---Silver in Sonic Mania Plus---

Created by The Vagabond
Special thanks Cylent Nite

How to install:
Open up your Sonic Mania directory (Program Files\Steam\steamapps\common\Sonic Mania)
and place the mod files in your mods folder
then make sure you have the mod manager installed.
Be sure to click install mods if you haven't already.
Once open, tick 'Silver in Sonic Mania Plus' and press save and play.
Have fun.
